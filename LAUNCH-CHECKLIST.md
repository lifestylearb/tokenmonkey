# TokenMonkey Launch — Step by Step

Everything is built. You just need to create 3 accounts and run a few commands.

---

## Step 1: GitHub (5 minutes)

You need a GitHub account to host the code publicly.

1. Go to https://github.com/signup — create an account (or use your existing one)
2. Create a new repository: https://github.com/new
   - Name: `tokenmonkey`
   - Set to **Public**
   - Click "Create repository"
3. Come back here and tell me the repo URL — I'll push all the code

---

## Step 2: npm (3 minutes)

npm is where developers download packages. Your SDK needs to be here.

1. Go to https://www.npmjs.com/signup — create an account
2. Back in your terminal, run:
   ```
   npm login
   ```
3. Tell me when you're logged in — I'll publish the SDK and MCP server

---

## Step 3: X / Twitter (5 minutes)

You need an account to post the launch thread.

1. Create @TokenMonkey (or whatever handle is available) at https://x.com
2. The launch thread is ready at `LAUNCH-THREAD.md` — just copy-paste each section as a tweet

---

## That's it.

Once you've done those 3 things, tell me and I'll:
- Push code to GitHub
- Publish SDK to npm
- Publish MCP server to npm
- Start the DegenMonkey demo bot to seed challenges
- Give you the exact tweets to post

Everything else is already built and ready.
